# sushi-sasha
# Суши от Саши — Ступино

**Современный премиум лендинг с корзиной и оформлением заказа**

Полноценный продающий сайт для доставки суши в Ступино, созданный с нуля с помощью вайбкодинга (Claude + Cursor).

### Что реализовано:
- Адаптивный премиум-дизайн в тёмной теме (в стиле Apple)
- Рабочая корзина с добавлением и управлением товарами
- Полноценная форма оформления заказа (доставка / самовывоз)
- Сохранение данных в localStorage
- Адаптивная вёрстка под все устройства

### Технологии:
- HTML5 + Tailwind CSS
- Vanilla JavaScript
- Vercel (деплой)

### Цель проекта:
Практика и развитие навыков быстрой разработки качественных веб-продуктов с помощью ИИ. 

Хочу продолжать улучшать проект, добавлять новые функции (анимации, слайдшоу, интеграции) и использовать его как кейс для дальнейшего роста в веб-разработке и вайбкодинге.
